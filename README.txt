Java style guide from: https://google.github.io/styleguide/javaguide.html

Program run through terminal using "gradle run",
as well as IntelliJ's "run" button.

JSON file: src/main/resources/default.json

Configuration file can change:
The location of the FinishLine.
The size, location and velocity of Stickman.
The number, position, and velocity of Clouds and Slimes.
The number and position of Platforms, Trees and Coins.

Testing:
Junit tests run using a JUnit configuration in IntelliJ.
Should also work with "gradle test" in terminal.

Game Assets sources:
https://startledpixels.itch.io/2d-pixel-item-asset-pack
https://ansimuz.itch.io/magic-cliffs-environment
